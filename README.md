# Cancer-Risk-Factor-Analysis_R
A data analysis project examining factors associated with cancer risk using R.

## Tools
- R
- R Markdown
- ggplot2
- dplyr

## Key Analysis
- Gender, Age and Cancer Risk Distribution
- Lifestyle Factor vs Cancer Risk
- Environmental Factor vs Cancer Risk
- Genetic Factor vs Cancer Risk

## Data Source
- Dataset: Cancer Risk Factors Dataset
- Source: [Kaggle](https://www.kaggle.com/datasets/tarekmasryo/cancer-risk-factors-dataset)
- Description: Contains 2,000 individual records with 21 factors influencing five common cancer types.
